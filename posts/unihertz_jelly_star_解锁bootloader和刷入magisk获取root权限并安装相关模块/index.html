Jelly Star 解锁 Bootloader

本文的操作均在 Ubuntu 操作系统下进行。首先安装ADB，打开Terminal,输入命令sudo apt-get install android-sdk-platform-tools 等待安装完成。然后打开手机 依次操作，Settings->About Phone->Build number。狂点 Build number ，直到出现 You are now a developer 。然后依次操作，Settings->System->Developer options。进入以后打开 Use developer options 的开关，找到 OEM unlocking 和 USB debugging 选项，打开开关。如下图所示：

然后用USB数据线链接电脑，在 Terminal 里输入adb devices，如果连接正确，该命令下方会现实你的设备。接着重启进入Bootloader模式。输入命令adb reboot Bootloader,接着继续输入fastboot flashing unlock。然后按音量+-键选择即可。

Jelly Star 刷入 Magisk 获取 Root 权限

首先下载安装 Magisk 到 Jelly Star。
下载好 Jelly Star 的系统固件包，然后解压出 boot.img 文件。将该文件放入 Jelly Star 手机中。然后到手机上操作，依次打开 Magsik 软件，如下图所示：

点击那个 install，然后出现下图：

选择刚才复制进手机的那个 boot.img 文件，然后出现下图：

接着将该操作生成的文件，复制到计算机。使用fastboot命令刷入该文件到手机。因为刚才进行了解锁 Bootloader 操作，系统恢复了出厂设置，故而需要重复进行一些操作，Settings->About Phone->Build number。狂点 Build number ，直到出现 You are now a developer 。然后依次操作，Settings->System->Developer options 。进入以后打开 Use developer options 的开关，找到 OEM unlocking 和 USB debugging 选项，打开开关。然后在 Terminal 输入adb devices，以检查连接正确。接着重启进入 Bootloader 模式。输入命令adb reboot Bootloader，然后刷入打了补丁的文件。使用命令fastboot flash boot /path/to/magisk_patched_[random_strings].img，回车等待结束，如果没有报错。接着重启手机，输入fastboot reboot。打开 Magisk 如果显示版本号，则已经成功获取了 Root 权限。

安装相关模块

安装LSPosed模块

下载LSPosed,在安装之前，需要去 Magisk 设置里打开 Zygisk 的开关。安装后重启手机，发现没有 LSPosed 的软件图标。你可以下拉通知栏，看到LSPosed的提示，点进去，在设置里边创建一个图标即可。然后你就可以安装很多有意思的模块了。
