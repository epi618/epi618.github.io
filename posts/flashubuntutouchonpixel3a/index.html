Pixel 3a刷入Ubuntu touch

PiAlpha included in Handheld Devices
 2023-10-15  898 words   2 minutes   views
Contents
  • 1 手机系统还原
  • 2 刷入 Ubuntu touch 系统

为什么要刷入 Ubuntu Touch 系统？ 还不是因为那个什么FanZha应用，该程序也许打着为你好的旗号，做些不为人知的事情。有些地方不安装该应用，不让孩子上学，有些地方有人强制要你安。为了推广该应用下了很大的力气，值得称赞。这下我用Ubuntu Touch，我总没法安装了吧！

为什么选择 Pixel 3a 来刷机，因为通过 Ubuntu Touch 官网，支持的最好，各项功能都能正常使用。

1 手机系统还原

将该手机系统还原至 Android 9 版本号为：PQ3B.190801.002 去 Pixel 的系统镜像官网下载。打开后如下图所示：

下载完成后，解压缩该包后得到 sargo-pq3b.190801.002 文件夹，进入文件夹查看，如下图所示：

flash-all.bat 适用于 Windows 操作系统；flash-all.sh适用于GNU/Linux系统。本文以 Ubuntu 操作系统为例。

首先打开手机上的开发者模式中的 USB 调试，然后查看 OEM 解锁是否打开，若没有，则先进行解锁 Bootloader 操作。接着 pixel 3a 用 USB 数据线连接到计算机。打开 Terminal 输入adb devices以检查手机是否正确连接。接着输入adb reboot bootloader,手机会自动重启进入 fastboot 模式

然后在 sargo-pq3b.190801.002 文件夹中，鼠标右键打开Terminal。在其中输入sudo ./flash-all.sh，即可看到开始刷写系统的过程信息。等待 Terminal 中的步骤完成。看到其中现实Finished 等待手机重启即可。

重启后进入手及查看其Android 版本和版本号。

2 刷入 Ubuntu touch 系统

打开 Ubuntu touch 的官网，找到pixe 3a的刷机页面。如下图所示。

下载 Debian 的刷机程序。在 Terminal 中将目录切换至你下载Debian 的刷机程序的目录下。输入sudo dpkg -i ubports-installer_0.10.0_linux_amd64.deb 然后等待安装完成。如出现报错，则需要安装相关依赖sudo apt install libappindicator1,然后重复上一步骤即可。

接着打开该刷机程序，如下图所示。

然后进入手机重新打开开发者模式，勾选 USB 调试，当出现要求信任调试的时候，确认即可。该程序会自动识别。如下图。

前文已经解锁 bootloader 和降级了Android 版本，直接点击 My device is unlocked 就好。然后出现如下图。

OS 的选择有 Ubuntu Touch 和 Doridian ，本文选择 Ubuntu Touch 然后点击 Install。 注意：当运行 UBports Installer 程序的时候需要开启全局上网模式，不然该程序会因下载不到文件而报错。

接下来就是耐心等待中，如下图。

刷机成功，等待手机重启，需要三分钟的样子。
