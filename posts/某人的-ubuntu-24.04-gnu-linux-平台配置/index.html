1 给ubuntu平台上的RTX 5060 Ti安装显卡驱动

1.1 更新内核

sudo add-apt-repository ppa:cappelikan/ppa #添加PPA
sudo apt update && sudo apt full-upgrade #更新
sudo apt install -y mainline #安装mainline
mainline list | less #查看linux内核版本
mainline install 6.14.4 #安装内核
sudo reboot #重启系统
uname -a #查看内核版本

1.2 安装GCC

sudo apt install gcc-14 #安装GCC
sudo update-alternatives --install /usr/bin/gcc gcc /usr/bin/gcc-14 14 
sudo update-alternatives --config gcc
gcc --version #查看GCC版本

1.3 安装nvidia驱动

输入命令行：

sudo ./NVIDIA-Linux-x86_64-575.51.02.run

选择MIT的版本，如果后续安装过程报错。则需要安装相关依赖库。命令行如下：

sudo apt install build-essential

然后重新运行安装即可。 检查驱动是否安装妥当，运行以下命令行：

nvidia-smi

2 安装输入法

先卸载 ubuntu 24.04 自带的 fcitx 4

sudo apt-get remove --purge fcitx
sudo apt-get autoremove

接着安装 fcitx5

sudo apt-get install fcitx5 fcitx5-configtool fcitx5-chinese-addons

然后打开 Fcitx Configuration，在 Input Method 中添加 Pinyin。接着在 Global Options 中的 Hotkey 设置Trigger Input Method为 Left Shift。

然后打开 .profile 文件， sudo nano ~/.profile ,在该文件的最后添加以下代码：

export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS=@im=fcitx
#fcitx5 &

接着给 Fcitx 安装词库。先下载词库 。解压文件后得到 CustomPinyinDictionary_Fcitx.dict 文件。然后将其复制到 /usr/share/fcitx5/pinyin/dictionaries/ 目录，如果没有自行创建。复制命令如下：

sudo cp CustomPinyinDictionary_Fcitx.dict /usr/share/fcitx5/pinyin/dictionaries/

重启即可。

3 安装 localsend

打开链接localsend,便于本地网络传输文件。下载适用的版本，此处我用的 ubuntu ，故而下载了 deb 的包。 然后在下载好的 deb 包的同一个目录下，打开 Terminal，输入：

sudo dpkg -i LocalSend-1.17.0-linux-x86-64.deb

如果报错，缺乏依赖，安装相关依赖后重新安装即可。

4安装 GoogleEarth

打开链接 Google Earth，下载适用的版本，此处我用的 ubuntu ，故而下载了 deb 的包。 然后在下载好的 deb 包的同一个目录下，打开 Terminal，输入：

sudo dpkg -i google-earth-pro-stable_7.3.6_amd64.deb

5 安装 Texstudio

添加 PPA，然后安装 Texstudio，命令如下：

sudo add-apt-repository ppa:sunderme/texstudio
sudo apt update
sudo apt install texstudio

6 安装 miniconda

打开中国科学技术大学的同步镜像网站，点击获取开源软件，选择 Miniconda ，下载即可。在该目录下打开 Terminal 。输入以下命令

sudo bash ./Miniconda3-latest-Linux-x86_64.sh

回车后，看完许可条款，然后输入 yes ，然后继续回车即可。

7 安装 gfortran

检查是否安装 gfortran ，命令如下：

which gfortran

如果没有则用以下命令行安装，

sudo apt install gfortran

安装完成后输入以下命令行，检查版本，以确认安装。

gfortran --version

8 安装 GMT (The Generic Mapping Tools)

因为 Ubuntu/Debian 官方的 GMT 二进制包不是最新版本的，故而采取编译源码安装的方式。

先安装相关依赖软件包，命令行如下：

#更新软件包列表
sudo apt update 
#安装必须软件包
sudo apt install build-essential cmake ninja-build libcurl4-gnutls-dev libnetcdf-dev gdal-bin libgdal-dev
sudo apt install libgeos-dev libglib2.0-dev libpcre3-dev libfftw3-dev liblapack-dev
sudo apt install ghostscript xdg-utils
#安装额外软件包
sudo apt install graphicsmagick ffmpeg

下载编译 GMT 所需要的三个文件：

GMT 6.5.0 源码：gmt-6.5.0-src.tar.xz

全球海岸线数据 GSHHG：gshhg-gmt-2.3.7.tar.gz

全球数字图表 DCW：dcw-gmt-2.1.2.tar.gz

然后将三个下载的文件解压，命令行如下：

#解压
tar -xvf gmt-6.5.0-src.tar.xz
tar -xvf gshhg-gmt-2.3.7.tar.gz
tar -xvf dcw-gmt-2.1.2.tar.gz
#移动
mv gshhg-gmt-2.3.7 gmt-6.5.0/share/gshhg-gmt
mv dcw-gmt-2.1.2 gmt-6.5.0/share/dcw-gmt
#切换目录
cd gmt-6.5.0
#编辑CMake文件
gedit cmake/ConfigUser.cmake

向打开的文件写入以下语句：

set (CMAKE_INSTALL_PREFIX "/opt/GMT-6.5.0")
set (GMT_USE_THREADS TRUE)

编译并且安装命令如下：

cmake --build .
sudo cmake --build . --target install

修改Shell配置文件，命令行如下：

gedit ~/.bashrc

在此文件的末尾添加以下语句。

export GMT6HOME=/opt/GMT-6.5.0
export PATH=${GMT6HOME}/bin:$PATH
export LD_LIBRARY_PATH=${LD_LIBRARY_PATH}:${GMT6HOME}/lib64

最后一步，检查是否安装成功。输入以下命令：

gmt --version

不出意外，应该会显示安装的版本号。

9 安装奥维互动地图

下载奥维互动地图的 Linux 版本，注意选择 CPU 的架构。下载过后是一个 deb 的包文件。在同一目录下，打开 Terminal ，直接使用命令：

sudo dpkg -i Linux-x86_64-OMap-10.1.8-build2025-01-22.3.deb

安装即可。

15 安装 VirtulBox

下载VirtualBox,然后使用以下命令：

sudo dpkg -i virtualbox-7.1_7.1.8-168469~Ubuntu~noble_amd64.deb

如果出现报错，则需要安装相关依赖软件包，命令行如下：

sudo apt install liblzf1 libqt6help6 libqt6statemachine6 libtpms0
sudo apt --fix-broken install

最后重新执行以下命令：

sudo dpkg -i virtualbox-7.1_7.1.8-168469~Ubuntu~noble_amd64.deb
